# Enter Tool

This module guides you on how to enter the canvas and design a project utilising various tools and entities.

## **Topics** 

- Toolbox
- Enter Canvas 
- Create Room 
- Room Settings 


### Toolbox

#### Enter Canvas Tools 

![floorplan](../static/img/enterTool/floorplan.png)  

**Floor Plan:** Use this option to upload pre-existing plans directly from BEATS.


![newfile](../static/img/enterTool/newfile.png)


**New File:** Select this option to start a project from scratch using pre-existing templates. 



![importfile](../static/img/enterTool/importfile.png)

**Import File:** With this option, you can import a plan directly from previous projects.

#### Project Canvas Tools 

**Navigation Tools**

![areaselection](../static/img/enterTool/areaselection.png)   **Area Selection Tool:** This tool is used for single or multiple module selections.

![moveoption](../static/img/enterTool/moveoption.png)  **Move Option Tool 2D:** This allows you to relocate or move any module or group of modules in a designated direction within a room. 

![rotate](../static/img/enterTool/rotate.png)  **Rotate Tool:** Used to rotate an entity in the room at any desirable angle. 

![perspective](../static/img/enterTool/perspective.png)  **Views / Perspective and Parallel Tool:** This tool provides default camera angles. It also defines the parallel and perspective views of the canvas.

![snap](../static/img/enterTool/snap.png)  **Snap Tool:** This tool will help in manoeuvrability, ensuring precise positioning of an entity adjacent to any other entity.  

**Trigger Tools**

![layers](../static/img/enterTool/layers.png)  **Layers Tool:** Dynamic layers help hide/unhide structural entities, while catalogue layers are categorised according to room settings.

![gallery](../static/img/enterTool/gallery.png)  **Gallery:** Captured images are stored and can be accessed from the gallery. 

![lights](../static/img/enterTool/lights.png)  **Lights:** To provide and add sources of illumination in the room. 

![settings](../static/img/enterTool/settings.png) **Settings:** Access complete room property settings. 

![export](../static/img/enterTool/export.png) **Export:** To export 2D drawings and a cut list. 

![import](../static/img/enterTool/import.png) **Import:** To import files into the project. They can be in JPG format or 3D model and can be imported from an existing project. 

![warehouse](../static/img/enterTool/warehouse.png) ![warehouse2](../static/img/enterTool/warehouse2.png)  **Warehouse:** This allows you to add modules and items in 3D for visualisation or quotation. 


**Design Tools**

![createroom](../static/img/enterTool/createroom.png)  **Create Room Shape:** Create a room using predefined shapes and templates, or create a custom shape for your plan. 

![createstructure](../static/img/enterTool/createstructure.png) **Create Structural Entities:**  Add structural entities like columns/beams/ slabs to your floor plan. This tool allows you to alter the properties of your entity. 

![windows](../static/img/enterTool/windows.png) **Doors/Windows Tool:** Add doors and windows to your floor plan.

![undo](../static/img/enterTool/undo.png) **Undo:** Directs the user to the previous action. 

![redo](../static/img/enterTool/redo.png) **Redo:** Reverse or repeat a previously undone action. 

![visualisation](../static/img/enterTool/visualisation.png) **Visualisation tool:** Adds visually appealing features like paint, wallpaper and flooring textures.

![measurement](../static/img/enterTool/measurement.png) **Measurement Tool:** Measures the distance between two points. 

![camera](../static/img/enterTool/camera.png) **Camera:** Captures images of the 3D, room-wise, from desired angles.

![environment](../static/img/enterTool/environment.png) **Environment Settings:** Adds and alters the exterior look of a project. 

![lit](../static/img/enterTool/lit.png) **Lit/Unlit Tool:** Provides a rendered view of the set view.

![text](../static/img/enterTool/text.png) **Text Tool:** Adds reference text in 2D or 3D canvas. 



### Enter Canvas 

To enter the canvas, the designer must undertake the following steps: 

**Step 1:** Click on the ![entercanvas](../static/img/enterTool/entercanvas.png) icon to enter the canvas tool screen. 

**Step 2:** Once you have entered the canvas, you must choose from three options. 

**Namely:** Floor Plan, New File, and Import File.* 

![options](../static/img/enterTool/options.png)

**Step 3:**  Based on your selection, for example:- To start a new project, select the ‘New File’ option to begin your journey. 

![newproject](../static/img/enterTool/newproject.png)

**Step 4:** Clicking on ‘Blank’ directs you to a new canvas showcasing several tool icons to start your design journey.

![blankcanvas](../static/img/enterTool/blankcanvas.png)



### Create Room 

This module guides you in creating a room using the tools available on your canvas.

**Step 1:** Click on the ‘Create Room’ ![createroom](../static/img/enterTool/createroom.png)  icon. 

**Step 2:** Select from the desired shapes and template options from the dropdown menu.  
![shapeoptions](../static/img/enterTool/shapeoptions.png)


:::tip
By clicking on the custom wall option in ‘shapes,’ you can create a custom shape for the room. 

- Drag and drop the chosen shape on the canvas to start your journey.  ![customwall](../static/img/enterTool/customwall.png)
:::

**Step 3:** Select the required room type, add the Room Name, and click submit to save the room.  

![roomtype](../static/img/enterTool/roomtype.png)


**Step 4:** Choose from 2D & 3D layouts featured on the toolbar using the toggle for preferred visualisation.* 


**2D Layout-**

![2Dlayout](../static/img/enterTool/2Dlayout.png)

**3D Layout-**

![3Dlayout](../static/img/enterTool/3Dlayout.png)

:::note
**Please note that the tool icons will vary per the selected layout.** 
:::

**For more information on the Navigation tools, refer to the TOOLBOX.**

![toolbox](../static/img/enterTool/toolbox.png)

**Step 5:** Start your design journey by clicking on the **"Structural Entities"** Tool icon

![structure](../static/img/enterTool/structure.png)


![structureentities](../static/img/enterTool/structureentities.png)

Alter the properties of the featured entities according to your preference.  
You have the option to add a Wall, Beam, Column or Slab from the dropdown menu. 

- On adding the entity, a properties bar on the right side of your screen will be visible, which allows one to add preferred dimensions and sizes. 

![properties](../static/img/enterTool/properties.png)

**Step 8:** Navigate towards the **"Door and Windows”** tool icon after finalising the structural entities. You may select the doors and windows' properties as per client requirements from the dropdown menu. 

![doors](../static/img/enterTool/doors.png)

- Move the cursor to place the selected feature. 

![selectfeature](../static/img/enterTool/selectfeature.png)

**Step 9:** You can review the room’s floor plan after establishing the structural and foundational features. 


### Room Settings 

Select and finalise the room material to be imported by defining its properties. 

**Step 1:** Click on the ![settings](../static/img/enterTool/settings.png) icon featured on the toolbar to access room settings. 

**Step 2:** Navigate the categories based on the selected catalogue. 

**For e.g-** If the selected room type is the kitchen, its catalogue will look like this: 

![catalogue](../static/img/enterTool/catalogue.png)

**Carcass:** Select the core material and brand from the drop-down menu along with the desired colour. 

![carcass](../static/img/enterTool/carcass.png)



**Shutter:** Choose from the options in the dropdown menu and select the shutter colour based on the client’s preference. 

![shutter](../static/img/enterTool/shutter.png)

**Handles:** Decide upon the handle design and finish used for cabinets. 

![handles](../static/img/enterTool/handles.png)

**For example,** 

**Edgebands:** Choose Edgebands from the options given below. 

![edgebands](../static/img/enterTool/edgebands.png)

**Hinges and Channels:** Select your hinges and channels from the dropdown menu.

![hinges](../static/img/enterTool/hinges.png)

:::note
*Edgeband and Hinges are features used for a complete room, not category-wise.
:::
